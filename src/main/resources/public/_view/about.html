How to access this site:

View URLS (access not logged publicly):

  • https://xss.to
  • https://xss.to/#path/{YOUR_PATH}, https://xss.to/#{...}
    • Remember that web browsers do not send the URL hash tag ('#') and the text after it to the server. This URL is still to the site's homepage (not logged). The client, in Javascript, interprets the location hash to filter your results or load other site information like this page.
  • http://hdryforlegnugar5.onion
    • If you have the TOR browser installed, you can anonymously browse the server over the Tor network at the above onion URL.
    • If you forget the onion domain name, there is a "tor" TXT record in the site's DNS:
      nslookup -q=txt xss.to
  • https://xss.to_{...}
    • The website internally uses URLs that start with underscores. Access to these URLs as well as favicon.ico are not logged.

Logged URLs:

  • https://xss.to/{YOUR_PATH}
    • Path must be at least 2 characters long and cannot start with an underscore.
  • https://xss.to/{YOUR_PATH}.png
    • Logged just like URLs that don't end in ".png", but the result returned to the client browser is a valid 1x1 PNG image.
